## 🧑‍💻 Getting Started Locally

```bash```
# Clone the repository
git clone https://github.com/Mus-1kan/MetaVerse-Fashion

# Go into the directory
cd metaverse-fashion

# Install dependencies
npm install

# Start the development server
npm run dev
